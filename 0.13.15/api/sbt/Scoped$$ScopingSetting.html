sbt.Scoped

ScopingSetting

sealed trait ScopingSetting[Result] extends AnyRef

Mixin trait for adding convenience vocabulary associated with specifiying the Scope of a setting. Allows specification of the Scope or part of the Scope of a setting being referenced.

Example: 

  1. name in Global := "hello Global scope"

name in (Compile, packageBin) := "hello Compile scope packageBin"

name in Compile := "hello Compile scope"

name.in(Compile).:=("hello ugly syntax")
