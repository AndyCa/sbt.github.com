CollectTypeTraverser

class CollectTypeTraverser[T] extends scala.tools.nsc.Global.TypeTraverser

Traverses given type and collects result of applying a partial function pf.

NOTE: This class exists in Scala 2.10 as CollectTypeCollector but does not in earlier versions (like 2.9) of Scala compiler that incremental cmpiler supports so we had to reimplement that class here.
