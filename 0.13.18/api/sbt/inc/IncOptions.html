sbt.inc

IncOptions

final class IncOptions extends Product with Serializable

Represents all configuration options for the incremental compiler itself and not the underlying Java/Scala compiler.

NOTE: This class used to be a case class but due to problems with retaining binary compatibility while new fields are added it has been expanded to a regular class. All compiler-generated methods for a case class has been defined explicitly.
